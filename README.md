# VEDICI Platform s.r.o. - Landing Page

Prezentačná webová stránka pre VEDICI Platform s.r.o., spoločnosť zameranú na vývoj a predaj chytrých suplementov pre e-gamerov.

## O projekte

Jednoduchá, moderná landing page poskytujúca základné informácie o spoločnosti a odkazujúca na e-shop [vedici.gg](https://vedici.gg).

## Štruktúra

- **Hero sekcia** - Úvodná sekcia s názvom spoločnosti a hlavným odkazom na e-shop
- **O nás** - Informácie o spoločnosti a jej poslaní
- **Produkty** - Predstavenie hlavných produktov (Game ON, Game CHANGER, Game OVER)
- **Kontakt** - Kontaktné informácie spoločnosti
- **Footer** - Zápatie stránky

## Technológie

- HTML5
- CSS3 (s modernými CSS premennými a flexbox/grid)
- Responzívny dizajn
- Google Fonts (Inter)

## Spustenie

Stránku môžete otvoriť priamo v prehliadači otvorením súboru `index.html`, alebo ju hostovať pomocou GitHub Pages.

## GitHub Pages

Pre aktiváciu GitHub Pages:
1. Prejdite do Settings repozitára
2. V sekcii Pages vyberte source branch (napr. main)
3. Stránka bude dostupná na `https://jurajgiacko.github.io/vedici-platform-web/`
